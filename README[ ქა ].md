# Fluminum (v2.2): C++-рЃќрЃћ рЃЏрЃљрЃбрЃарЃўрЃфрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃЏрЃљрЃдрЃљрЃџрЃў рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃўрЃА рЃЏрЃгрЃЋрЃћрЃарЃЋрЃљрЃџрЃў ­Ъџђ

рЃЎрЃћрЃЌрЃўрЃџрЃў рЃўрЃДрЃЮрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЏрЃЮрЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ **Fluminum**-рЃерЃў РђЊ рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџ C++ рЃЏрЃФрЃџрЃљрЃЋрЃа рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљрЃерЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃЏрЃљрЃбрЃарЃўрЃфрЃБрЃџрЃў рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃћрЃЉрЃўрЃА рЃАрЃўрЃЕрЃЦрЃљрЃарЃўрЃА рЃАрЃљрЃќрЃдрЃЋрЃарЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃарЃдрЃЋрЃћрЃЋрЃљрЃЊ. рЃћрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃљрЃа рЃљрЃарЃўрЃА рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃЏрЃЮрЃарЃўрЃњрЃў рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ; рЃћрЃА рЃљрЃарЃўрЃА **рЃерЃбрЃарЃљрЃАрЃћрЃюрЃўрЃА рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃўрЃА**, **`std::async`**-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃБрЃџрЃў **рЃЏрЃарЃљрЃЋрЃљрЃџрЃюрЃљрЃЎрЃљрЃЊрЃўрЃљрЃюрЃЮрЃЉрЃўрЃА**, рЃЊрЃљрЃЉрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА **SIMD (AVX/SSE2) рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃћрЃЉрЃўрЃАрЃљ** рЃЊрЃљ рЃўрЃюрЃбрЃћрЃџрЃћрЃЦрЃбрЃБрЃљрЃџрЃБрЃарЃў **рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃќрЃћ рЃЏрЃЮрЃарЃњрЃћрЃЉрЃБрЃџрЃў рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА** рЃАрЃўрЃюрЃћрЃарЃњрЃўрЃБрЃџрЃў рЃФрЃљрЃџрЃўрЃА рЃЊрЃљрЃАрЃбрЃБрЃарЃў.

рЃЊрЃљрЃўрЃЋрЃўрЃгрЃДрЃћрЃЌ $O(N^3)$ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃюрЃћрЃџрЃў рЃЊрЃдрЃћрЃћрЃЉрЃў. Fluminum рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА **рЃњрЃљрЃАрЃљрЃЮрЃфрЃљрЃа рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃўрЃА рЃќрЃарЃЊрЃљрЃА, 574x-рЃЏрЃЊрЃћ рЃљрЃЕрЃЦрЃљрЃарЃћрЃЉрЃўрЃЌ** рЃбрЃарЃљрЃЊрЃўрЃфрЃўрЃБрЃџ рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃЌрЃљрЃю рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃЌ рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃљрЃќрЃћ. рЃЏрЃљрЃА рЃњрЃљрЃљрЃЕрЃюрЃўрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃќрЃћ рЃЏрЃЮрЃарЃњрЃћрЃЉрЃБрЃџрЃў рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃў рЃЎрЃЮрЃюрЃАрЃЮрЃџрЃўрЃА рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў, рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў рЃърЃарЃЮрЃњрЃарЃћрЃАрЃўрЃА рЃќрЃЮрЃџрЃћрЃЉрЃў, рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃўрЃА рЃљрЃюрЃљрЃџрЃўрЃбрЃўрЃЎрЃљ рЃЊрЃљ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃФрЃџрЃљрЃЋрЃарЃў рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ, рЃарЃљрЃф рЃЏрЃљрЃА рЃљрЃЦрЃфрЃћрЃЋрЃА рЃарЃЮрЃњрЃЮрЃарЃф рЃЏрЃФрЃџрЃљрЃЋрЃа рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃљрЃЊ, рЃљрЃАрЃћрЃЋрЃћ рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃБрЃџ рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљрЃЊ.

## Рюе рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў

  * **РџА рЃћрЃџрЃЋрЃўрЃАрЃћрЃЉрЃБрЃарЃў рЃАрЃўрЃЕрЃЦрЃљрЃарЃћ:** рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА рЃерЃбрЃарЃљрЃАрЃћрЃюрЃўрЃА $O(N^{2.807})$ рЃАрЃўрЃарЃЌрЃБрЃџрЃћрЃА рЃдрЃарЃЏрЃљ рЃърЃљрЃарЃљрЃџрЃћрЃџрЃўрЃќрЃЏрЃЌрЃљрЃю рЃЊрЃљ SIMD-рЃЌрЃљрЃю рЃерЃћрЃБрЃЊрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
  * **­ЪДа рЃўрЃюрЃбрЃћрЃџрЃћрЃЦрЃбрЃБрЃљрЃџрЃБрЃарЃў рЃерЃбрЃарЃљрЃАрЃћрЃюрЃў:** рЃљрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃћрЃЉрЃА **рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃБрЃџ, рЃърЃљрЃарЃљрЃџрЃћрЃџрЃБрЃа рЃерЃбрЃарЃљрЃАрЃћрЃюрЃўрЃА рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃА** **рЃљрЃЊрЃљрЃърЃбрЃўрЃарЃћрЃЉрЃљрЃЊрЃў рЃќрЃдрЃЋрЃарЃБрЃџрЃў** рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃЌ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЊрЃљрЃЊрЃўрЃА SIMD-рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃћрЃЉрЃБрЃџ рЃюрЃљрЃўрЃЋрЃБрЃа рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљрЃќрЃћ рЃЏрЃфрЃўрЃарЃћ рЃЦрЃЋрЃћрЃљрЃЏрЃЮрЃфрЃљрЃюрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃЮрЃЉрЃўрЃА рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА.
  * **­Ъїљ рЃЏрЃарЃљрЃЋрЃљрЃџрЃЉрЃўрЃарЃЌрЃЋрЃўрЃљрЃюрЃў рЃЮрЃАрЃбрЃљрЃбрЃЮрЃЉрЃљ:** рЃўрЃДрЃћрЃюрЃћрЃЉрЃА `std::async`-рЃА рЃЊрЃљ `std::future`-рЃА рЃерЃбрЃарЃљрЃАрЃћрЃюрЃўрЃА 7 рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃБрЃџрЃў рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃўрЃА рЃЊрЃљ рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃА рЃљрЃЏрЃЮрЃфрЃљрЃюрЃћрЃЉрЃўрЃА **рЃЊрЃўрЃюрЃљрЃЏрЃўрЃБрЃарЃљрЃЊ рЃњрЃљрЃАрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃџрЃљрЃЊ** рЃДрЃЋрЃћрЃџрЃљ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏ CPU рЃЉрЃўрЃарЃЌрЃЋрЃќрЃћ.
  * **­ЪЈј№ИЈ SIMD рЃљрЃЕрЃЦрЃљрЃарЃћрЃЉрЃљ:** рЃАрЃљрЃЉрЃљрЃќрЃўрЃАрЃЮ (рЃюрЃљрЃўрЃЋрЃБрЃарЃў) рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљ рЃњрЃљрЃФрЃџрЃўрЃћрЃарЃћрЃЉрЃБрЃџрЃўрЃљ **AVX рЃЊрЃљ SSE2 рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃўрЃЌ**, рЃарЃљрЃф рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃљрЃФрЃџрЃћрЃЋрЃА CPU-рЃА рЃЊрЃљрЃљрЃЏрЃБрЃерЃљрЃЮрЃА рЃЏрЃарЃљрЃЋрЃљрЃџрЃў `double` рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃћрЃарЃЌ рЃфрЃўрЃЎрЃџрЃерЃў, рЃАрЃљрЃЊрЃљрЃф рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃљ рЃљрЃЏрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃўрЃФрЃџрЃћрЃЋрЃљ.
  * **­ЪќЦ№ИЈ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃќрЃћ рЃЏрЃЮрЃарЃњрЃћрЃЉрЃБрЃџрЃў:** рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ **рЃљрЃЏрЃЮрЃўрЃфрЃюрЃЮрЃЉрЃА CPU рЃЉрЃўрЃарЃЌрЃЋрЃћрЃЉрЃА, рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏ рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃљрЃА рЃЊрЃљ SIMD рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљрЃА** (AVX/SSE2), рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃћрЃЉрЃАрЃљ рЃЊрЃљ рЃњрЃљрЃцрЃарЃЌрЃ«рЃўрЃџрЃћрЃЉрЃћрЃЉрЃА.
  * **­ЪЊі рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃў рЃЊрЃљ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃБрЃџрЃў:** рЃњрЃљрЃљрЃЕрЃюрЃўрЃљ **рЃЏрЃЊрЃўрЃЊрЃљрЃарЃў рЃЎрЃЮрЃюрЃАрЃЮрЃџрЃўрЃА UI** рЃцрЃћрЃарЃћрЃЉрЃўрЃЌ, рЃЕрЃљрЃарЃЕрЃЮрЃћрЃЉрЃўрЃЌ, рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃБрЃарЃў рЃърЃарЃЮрЃњрЃарЃћрЃАрЃўрЃА рЃќрЃЮрЃџрЃўрЃЌ, рЃ«рЃЏрЃЮрЃЋрЃљрЃюрЃў рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃўрЃЌ, рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃўрЃА рЃљрЃюрЃљрЃџрЃўрЃќрЃўрЃЌ (QPC рЃЊрЃљ Chrono рЃбрЃљрЃўрЃЏрЃћрЃарЃћрЃЉрЃўрЃА рЃЕрЃљрЃЌрЃЋрЃџрЃўрЃЌ) рЃЊрЃљ **CSV рЃџрЃЮрЃњрЃўрЃарЃћрЃЉрЃўрЃЌ**.
  * **­ЪЏа№ИЈ рЃЏрЃбрЃЎрЃўрЃфрЃћ рЃЊрЃљ рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃў:** рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА **рЃљрЃарЃљрЃ«рЃљрЃарЃўрЃАрЃ«рЃЮрЃЋрЃљрЃюрЃў рЃЮрЃарЃў рЃЊрЃљ рЃљрЃарЃљрЃЎрЃЋрЃљрЃЊрЃарЃљрЃбрЃБрЃџрЃў рЃЏрЃљрЃбрЃарЃўрЃфрЃћрЃЉрЃА** рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃерЃћрЃЋрЃАрЃћрЃЉрЃўрЃЌ/рЃЏрЃЮрЃГрЃарЃўрЃЌ, рЃњрЃЌрЃљрЃЋрЃљрЃќрЃЮрЃЉрЃЌ рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЏрЃћрЃЌрЃЮрЃЊрЃА (рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌрЃў, рЃЎрЃЮрЃюрЃАрЃЮрЃџрЃў, рЃцрЃљрЃўрЃџрЃў) рЃЊрЃљ рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА.
  * **­ЪћЇ рЃърЃљрЃарЃљрЃџрЃћрЃџрЃБрЃарЃў рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃљ:** рЃњрЃЌрЃљрЃЋрЃљрЃќрЃЮрЃЉрЃЌ рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃБрЃџ, рЃърЃљрЃарЃљрЃџрЃћрЃџрЃБрЃа рЃЏрЃљрЃбрЃарЃўрЃфрЃўрЃА рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃА **epsilon рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃўрЃЌ** рЃЏрЃфрЃЮрЃфрЃљрЃЋ-рЃЏрЃФрЃўрЃЏрЃўрЃљрЃюрЃў рЃАрЃўрЃќрЃБрЃАрЃбрЃўрЃА рЃерЃћрЃАрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃџрЃљрЃЊ.
  * **­ЪЊЮ рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ C++:** рЃљрЃњрЃћрЃЉрЃБрЃџрЃўрЃљ C++17-рЃќрЃћ, рЃљрЃЦрЃфрЃћрЃюрЃбрЃўрЃЌ рЃАрЃБрЃцрЃЌрЃљ рЃЎрЃЮрЃЊрЃќрЃћ, RAII-рЃќрЃћ рЃЊрЃљ рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃЎрЃЮрЃюрЃЎрЃБрЃарЃћрЃюрЃбрЃБрЃџрЃЮрЃЉрЃўрЃА рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃќрЃћ.

-----

## ­ЪњА рЃерЃбрЃарЃљрЃАрЃћрЃюрЃўрЃА рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃў: "рЃЊрЃљрЃДрЃљрЃЋрЃў рЃЊрЃљ рЃўрЃЉрЃљрЃбрЃЮрЃюрЃћ" рЃарЃћрЃЋрЃЮрЃџрЃБрЃфрЃўрЃљ

рЃЏрЃљрЃбрЃарЃўрЃфрЃћрЃЉрЃўрЃА рЃбрЃарЃљрЃЊрЃўрЃфрЃўрЃБрЃџрЃў рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљ рЃљрЃарЃўрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃарЃў, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃЌрЃљрЃЊ рЃўрЃюрЃбрЃћрЃюрЃАрЃўрЃБрЃарЃў рЃърЃарЃЮрЃфрЃћрЃАрЃў рЃЎрЃБрЃЉрЃБрЃарЃў рЃАрЃўрЃарЃЌрЃБрЃџрЃўрЃЌ ($O(N^3)$). рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ рЃЏрЃљрЃбрЃарЃўрЃфрЃўрЃА рЃќрЃЮрЃЏрЃўрЃА рЃњрЃљрЃЮрЃарЃЏрЃљрЃњрЃћрЃЉрЃљ рЃАрЃљрЃЏрЃБрЃерЃљрЃЮрЃА рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃљрЃА рЃарЃЋрЃљрЃ»рЃћрЃа рЃќрЃарЃЊрЃўрЃА, рЃарЃљрЃф рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃ«рЃЊрЃћрЃЉрЃљ рЃљрЃарЃљрЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃЊрЃўрЃЊрЃў рЃЏрЃљрЃбрЃарЃўрЃфрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

рЃЋрЃЮрЃџрЃЎрЃћрЃа рЃерЃбрЃарЃљрЃАрЃћрЃюрЃўрЃА 1969 рЃгрЃџрЃўрЃА рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃЏрЃљ рЃерЃћрЃфрЃЋрЃљрЃџрЃљ рЃЌрЃљрЃЏрЃљрЃерЃўрЃА рЃгрЃћрЃАрЃћрЃЉрЃў. рЃўрЃА рЃўрЃДрЃћрЃюрЃћрЃЉрЃА "рЃЊрЃљрЃДрЃљрЃЋрЃў рЃЊрЃљ рЃўрЃЉрЃљрЃбрЃЮрЃюрЃћ" рЃАрЃбрЃарЃљрЃбрЃћрЃњрЃўрЃљрЃА:

1.  **рЃЊрЃљрЃДрЃЮрЃцрЃљ:** рЃЊрЃўрЃЊрЃў $N \\times N$ рЃЏрЃљрЃбрЃарЃўрЃфрЃўрЃА рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃцрЃљ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃљрЃЊ рЃЏрЃфрЃўрЃарЃћ $N/2 \\times N/2$ рЃЦрЃЋрЃћ-рЃЏрЃљрЃбрЃарЃўрЃфрЃћрЃЉрЃќрЃћ.
2.  **рЃЊрЃљрЃърЃДрЃарЃЮрЃЉрЃљ (рЃ«рЃарЃўрЃЎрЃў):** рЃюрЃљрЃўрЃЋрЃБрЃарЃў рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃўрЃЌ рЃАрЃљрЃГрЃўрЃарЃЮ 8 рЃЦрЃЋрЃћ-рЃЏрЃљрЃбрЃарЃўрЃфрЃўрЃА рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃўрЃА рЃюрЃљрЃфрЃЋрЃџрЃљрЃЊ, рЃерЃбрЃарЃљрЃАрЃћрЃюрЃў рЃГрЃЎрЃЋрЃўрЃљрЃюрЃБрЃарЃљрЃЊ рЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃћрЃЉрЃА рЃўрЃАрЃћ, рЃарЃЮрЃЏ рЃАрЃљрЃГрЃўрЃарЃЮ рЃњрЃљрЃ«рЃЊрЃћрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ **7 рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљ**. рЃћрЃА рЃЏрЃўрЃўрЃдрЃгрЃћрЃЋрЃљ 10 рЃерЃБрЃљрЃџрЃћрЃЊрЃБрЃарЃў рЃ»рЃљрЃЏрЃў/рЃАрЃ«рЃЋрЃљрЃЮрЃЉрЃљ рЃЏрЃљрЃбрЃарЃўрЃфрЃўрЃА (S1-S10) рЃерЃћрЃЏрЃЮрЃдрЃћрЃЉрЃўрЃЌ рЃЊрЃљ 7 рЃюрЃљрЃЏрЃарЃљрЃЋрЃџрЃў рЃЏрЃљрЃбрЃарЃўрЃфрЃўрЃА (P1-P7) рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃЌ.
3.  **рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃљ:** 7 рЃюрЃљрЃЏрЃарЃљрЃЋрЃџрЃў рЃЏрЃљрЃбрЃарЃўрЃфрЃљ рЃћрЃарЃЌрЃўрЃљрЃюрЃЊрЃћрЃЉрЃљ (рЃерЃћрЃЎрЃарЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃўрЃА рЃњрЃќрЃўрЃЌ), рЃарЃљрЃЌрЃљ рЃерЃћрЃўрЃЦрЃЏрЃюрЃљрЃА рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃерЃћрЃЊрЃћрЃњрЃўрЃА рЃЏрЃљрЃбрЃарЃўрЃфрЃўрЃА рЃЮрЃЌрЃ«рЃў рЃЎрЃЋрЃљрЃЊрЃарЃљрЃюрЃбрЃў.

$$A = \begin{bmatrix} A_{11} & A_{12} \\ A_{21} & A_{22} \end{bmatrix}, \quad B = \begin{bmatrix} B_{11} & B_{12} \\ B_{21} & B_{22} \end{bmatrix}, \quad C = \begin{bmatrix} C_{11} & C_{12} \\ C_{21} & C_{22} \end{bmatrix}$$

  * $S\_1 = B\_{12} - B\_{22}$
  * $S\_2 = A\_{11} + A\_{12}$
  * $S\_3 = A\_{21} + A\_{22}$
  * $S\_4 = B\_{21} - B\_{11}$
  * $S\_5 = A\_{11} + A\_{22}$
  * $S\_6 = B\_{11} + B\_{22}$
  * $S\_7 = A\_{12} - A\_{22}$
  * $S\_8 = B\_{21} + B\_{22}$
  * $S\_9 = A\_{11} - A\_{21}$
  * $S\_{10} = B\_{11} + B\_{12}$
  * $P\_1 = S\_5 \\times S\_6$
  * $P\_2 = S\_3 \\times B\_{11}$
  * $P\_3 = A\_{11} \\times S\_1$
  * $P\_4 = A\_{22} \\times S\_4$
  * $P\_5 = S\_2 \\times B\_{22}$
  * $P\_6 = S\_9 \\times S\_{10}$
  * $P\_7 = S\_7 \\times S\_8$
  * $C\_{11} = P\_1 + P\_4 - P\_5 + P\_7$
  * $C\_{12} = P\_3 + P\_5$
  * $C\_{21} = P\_2 + P\_4$
  * $C\_{22} = P\_1 - P\_2 + P\_3 + P\_6$

рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃўрЃА рЃћрЃА рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃБрЃџрЃў рЃерЃћрЃЏрЃфрЃўрЃарЃћрЃЉрЃљ 8-рЃЊрЃљрЃю 7-рЃЏрЃЊрЃћ рЃўрЃгрЃЋрЃћрЃЋрЃА $O(N^{2.807})$ рЃАрЃўрЃарЃЌрЃБрЃџрЃћрЃА. рЃЏрЃўрЃБрЃ«рЃћрЃЊрЃљрЃЋрЃљрЃЊ рЃўрЃЏрЃўрЃАрЃљ, рЃарЃЮрЃЏ рЃерЃћрЃЎрЃарЃћрЃЉрЃћрЃЉрЃўрЃА/рЃњрЃљрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљ рЃўрЃќрЃарЃЊрЃћрЃЉрЃљ, рЃЊрЃўрЃЊрЃў N-рЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃфрЃўрЃарЃћрЃЉрЃљ рЃЊрЃЮрЃЏрЃўрЃюрЃўрЃарЃћрЃЉрЃА, рЃарЃљрЃф рЃўрЃгрЃЋрЃћрЃЋрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃю рЃљрЃЕрЃЦрЃљрЃарЃћрЃЉрЃљрЃА. Fluminum рЃЏрЃљрЃарЃЌрЃљрЃЋрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃ«рЃљрЃарЃ»рЃћрЃЉрЃА **SIMD-рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃћрЃЉрЃБрЃџ рЃюрЃљрЃўрЃЋрЃБрЃа рЃЏрЃћрЃЌрЃЮрЃЊрЃќрЃћ рЃњрЃљрЃЊрЃљрЃарЃЌрЃЋрЃўрЃЌ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃћрЃЉрЃљрЃЊ рЃќрЃдрЃЋрЃљрЃарЃќрЃћ рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ**, рЃарЃљрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃа рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃљрЃА рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃљрЃАрЃерЃбрЃљрЃЉрЃерЃў.

-----

## РџЎ№ИЈ Fluminum-рЃўрЃА рЃБрЃърЃўрЃарЃљрЃбрЃћрЃАрЃЮрЃЉрЃљ: рЃдрЃарЃЏрЃљ рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃћрЃЉрЃў

Fluminum рЃљрЃдрЃгрЃћрЃЋрЃА рЃЌрЃљрЃЋрЃўрЃА рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃљрЃА рЃЏрЃарЃљрЃЋрЃљрЃџрЃЏрЃ«рЃарЃўрЃЋрЃў рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃАрЃбрЃарЃљрЃбрЃћрЃњрЃўрЃўрЃЌ:

1.  **рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃБрЃџрЃў рЃърЃљрЃарЃљрЃџрЃћрЃџрЃўрЃќрЃЏрЃў (`std::async`)**: рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ `strassen_recursive_worker` рЃљрЃарЃўрЃА рЃњрЃБрЃџрЃерЃў. рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃЏрЃљрЃбрЃарЃўрЃфрЃљ рЃўрЃДрЃЮрЃцрЃљ, 7 рЃерЃћрЃЏрЃЊрЃњрЃЮрЃЏрЃў рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃБрЃџрЃў рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃљ (P1-P7) рЃњрЃљрЃўрЃерЃЋрЃћрЃЉрЃљ рЃарЃЮрЃњрЃЮрЃарЃф **рЃљрЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃБрЃџрЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃћрЃЉрЃў `std::async`-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ**. рЃћрЃА рЃЦрЃЏрЃюрЃўрЃА рЃърЃљрЃарЃљрЃџрЃћрЃџрЃБрЃарЃў рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃћрЃЉрЃўрЃА рЃ«рЃћрЃА. рЃљрЃЏ рЃърЃљрЃарЃљрЃџрЃћрЃџрЃўрЃќрЃЏрЃўрЃА рЃАрЃўрЃдрЃарЃЏрЃћ (`max_depth_async`) рЃњрЃљрЃЏрЃЮрЃўрЃЌрЃЋрЃџрЃћрЃЉрЃљ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃў рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃБрЃџрЃў рЃюрЃљрЃЎрЃљрЃЊрЃћрЃЉрЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃћрЃџрЃќрЃћ, рЃарЃљрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃЉрЃўрЃарЃЌрЃЋрЃћрЃЉрЃўрЃА рЃЮрЃърЃбрЃўрЃЏрЃљрЃџрЃБрЃа рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃА рЃќрЃћрЃЊрЃЏрЃћрЃбрЃў рЃюрЃљрЃЎрЃљрЃЊрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ.
2.  **SIMD-рЃљрЃЕрЃЦрЃљрЃарЃћрЃЉрЃБрЃџрЃў рЃАрЃљрЃЉрЃљрЃќрЃўрЃАрЃЮ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљ (`multiply_naive`)**: рЃерЃбрЃарЃљрЃАрЃћрЃюрЃўрЃА рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃљ рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮрЃЊ рЃљрЃдрЃгрЃћрЃЋрЃА рЃАрЃљрЃЉрЃљрЃќрЃўрЃАрЃЮ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃА. рЃБрЃЉрЃарЃљрЃџрЃЮ рЃюрЃљрЃўрЃЋрЃБрЃарЃў рЃфрЃўрЃЎрЃџрЃўрЃА рЃюрЃљрЃфрЃЋрЃџрЃљрЃЊ, Fluminum-рЃўрЃА `multiply_naive` рЃўрЃДрЃћрЃюрЃћрЃЉрЃА **Single Instruction, Multiple Data (SIMD) рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃА**. рЃўрЃА рЃ»рЃћрЃа рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА **AVX** рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљрЃА, рЃЊрЃљ рЃЌрЃБ AVX рЃЏрЃўрЃБрЃгрЃЋрЃЊрЃЮрЃЏрЃћрЃџрЃўрЃљ, рЃњрЃљрЃЊрЃљрЃЊрЃўрЃА **SSE2**-рЃќрЃћ. рЃћрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃљрЃФрЃџрЃћрЃЋрЃА CPU-рЃА рЃерЃћрЃљрЃАрЃарЃБрЃџрЃЮрЃА рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃћрЃЉрЃў 4 (AVX) рЃљрЃю 2 (SSE2) `double` рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљрЃќрЃћ рЃћрЃарЃЌрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ, рЃарЃљрЃф рЃЏрЃЎрЃЋрЃћрЃЌрЃарЃљрЃЊ рЃљрЃЕрЃЦрЃљрЃарЃћрЃЉрЃА рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃарЃљрЃЋрЃљрЃџрЃарЃўрЃфрЃ«рЃЮрЃЋрЃљрЃю, рЃЏрЃфрЃўрЃарЃћ рЃЏрЃљрЃАрЃерЃбрЃљрЃЉрЃўрЃА рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃА.
3.  **рЃљрЃЊрЃљрЃърЃбрЃўрЃарЃћрЃЉрЃљрЃЊрЃў рЃќрЃдрЃЋрЃљрЃарЃў**: рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃЮрЃА `threshold`. рЃљрЃЏ рЃќрЃЮрЃЏрЃљрЃќрЃћ рЃЏрЃфрЃўрЃарЃћ рЃЏрЃљрЃбрЃарЃўрЃфрЃћрЃЉрЃў рЃЏрЃБрЃерЃљрЃЋрЃЊрЃћрЃЉрЃљ рЃАрЃгрЃарЃљрЃцрЃў `multiply_naive` рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃЌ, рЃарЃљрЃф рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃю рЃљрЃфрЃўрЃџрЃћрЃЉрЃА рЃерЃбрЃарЃљрЃАрЃћрЃюрЃўрЃА рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃўрЃАрЃљ рЃЊрЃљ рЃЏрЃљрЃбрЃарЃўрЃфрЃБрЃџрЃў рЃерЃћрЃЎрЃарЃћрЃЉрЃћрЃЉрЃўрЃА/рЃњрЃљрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃ«рЃљрЃарЃ»рЃћрЃЉрЃА, рЃАрЃљрЃЊрЃљрЃф рЃўрЃАрЃўрЃюрЃў рЃљрЃа рЃљрЃарЃўрЃА рЃАрЃљрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃЮ. рЃќрЃдрЃЋрЃарЃўрЃА 0-рЃќрЃћ рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ рЃљрЃўрЃФрЃБрЃџрЃћрЃЉрЃА рЃюрЃљрЃўрЃЋрЃБрЃа (рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЏрЃљрЃўрЃюрЃф SIMD-рЃљрЃЕрЃЦрЃљрЃарЃћрЃЉрЃБрЃџ) рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљрЃА.
4.  **рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃерЃћрЃЋрЃАрЃћрЃЉрЃљ**: рЃерЃбрЃарЃљрЃАрЃћрЃюрЃўрЃА рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃў (рЃарЃЮрЃњрЃЮрЃарЃф рЃљрЃЦ рЃљрЃарЃўрЃА рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў) рЃЏрЃЮрЃўрЃЌрЃ«рЃЮрЃЋрЃА рЃЎрЃЋрЃљрЃЊрЃарЃљрЃбрЃБрЃџ рЃЏрЃљрЃбрЃарЃўрЃфрЃћрЃЉрЃА, рЃарЃЮрЃЏрЃћрЃџрЃЌрЃљ рЃњрЃљрЃюрЃќрЃЮрЃЏрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў рЃЮрЃарЃўрЃА рЃ«рЃљрЃарЃўрЃАрЃ«рЃўрЃљ. Fluminum **рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃА рЃќрЃЮрЃЏрЃљрЃА** рЃЏрЃљрЃбрЃарЃўрЃфрЃћрЃЉрЃўрЃА рЃюрЃБрЃџрЃћрЃЉрЃўрЃЌ рЃерЃћрЃЋрЃАрЃћрЃЉрЃўрЃЌ рЃЏрЃЮрЃЏрЃЊрЃћрЃЋрЃюрЃЮ рЃЮрЃарЃўрЃА рЃ«рЃљрЃарЃўрЃАрЃ«рЃљрЃЏрЃЊрЃћ (`Matrix::pad`) рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљрЃЏрЃЊрЃћ рЃЊрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЏрЃљрЃЌ рЃЌрЃљрЃЋрЃЊрЃљрЃърЃўрЃарЃЋрЃћрЃџ рЃќрЃЮрЃЏрЃљрЃЏрЃЊрЃћ рЃЏрЃЮрЃГрЃарЃўрЃЌ (`Matrix::unpad`) рЃерЃћрЃЏрЃЊрЃњрЃЮрЃЏ.
5.  **рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃфрЃЮрЃЊрЃюрЃљ**: рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљрЃЏрЃЊрЃћ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏ RAM-рЃА рЃЊрЃљ рЃљрЃцрЃљрЃАрЃћрЃЉрЃА рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃърЃўрЃЎрЃБрЃа рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃА, рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃњрЃљрЃЏрЃЮрЃАрЃфрЃћрЃЏрЃА рЃњрЃљрЃцрЃарЃЌрЃ«рЃўрЃџрЃћрЃЉрЃћрЃЉрЃА. рЃўрЃА рЃўрЃДрЃћрЃюрЃћрЃЉрЃА `GlobalMemoryStatusEx`-рЃА рЃЊрЃљ `GetProcessMemoryInfo`-рЃА (Windows-рЃАрЃърЃћрЃфрЃўрЃцрЃўрЃЎрЃБрЃарЃў) рЃљрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА.
6.  **рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃўрЃџрЃћрЃЉрЃљрЃќрЃћ рЃцрЃЮрЃЎрЃБрЃАрЃўрЃарЃћрЃЉрЃљ**: рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ `display_progress` рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃфрЃљрЃџрЃЎрЃћ рЃюрЃљрЃЎрЃљрЃЊрЃерЃў рЃЊрЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА `std::atomic<int>` рЃЏрЃарЃўрЃфрЃ«рЃЋрЃћрЃџрЃА, рЃарЃљрЃЌрЃљ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃА **рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў рЃърЃарЃЮрЃњрЃарЃћрЃАрЃўрЃА рЃќрЃЮрЃџрЃў** рЃ«рЃљрЃюрЃњрЃарЃФрЃџрЃўрЃЋрЃў рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮрЃА. рЃЎрЃЮрЃюрЃАрЃЮрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃў рЃњрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃБрЃџрЃўрЃљ рЃцрЃћрЃарЃћрЃЉрЃўрЃЌрЃљ рЃЊрЃљ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃЕрЃљрЃарЃЕрЃЮрЃћрЃЉрЃўрЃЌ рЃБрЃЎрЃћрЃЌрЃћрЃАрЃў рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃЊрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

-----

## ­ЪЈЌ№ИЈ рЃЎрЃЮрЃЊрЃўрЃА рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ (`fluminumTversion.cpp`)

рЃћрЃарЃЌрЃў C++ рЃцрЃљрЃўрЃџрЃў `fluminumTversion.cpp` рЃљрЃарЃўрЃА рЃЊрЃљрЃЏрЃЮрЃБрЃЎрЃўрЃЊрЃћрЃЉрЃћрЃџрЃў рЃћрЃарЃЌрЃћрЃБрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃљрЃЏ рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃА.

  * **Includes**: рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃўрЃА рЃАрЃљрЃЌрЃљрЃБрЃарЃћрЃЉрЃўрЃА (`<vector>`, `<thread>`, `<future>`, `<atomic>`, `<chrono>`, `<immintrin.h>`) рЃЊрЃљ Windows-рЃАрЃърЃћрЃфрЃўрЃцрЃўрЃЎрЃБрЃарЃў рЃАрЃљрЃЌрЃљрЃБрЃарЃћрЃЉрЃўрЃА (`<windows.h>`, `<psapi.h>`) рЃАрЃарЃБрЃџрЃў рЃюрЃљрЃЎрЃарЃћрЃЉрЃў.
  * **рЃЎрЃЮрЃюрЃАрЃЮрЃџрЃўрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃљ**: рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА (`print_header_box`, `print_line_in_box`) рЃЊрЃљ рЃЎрЃЮрЃюрЃАрЃбрЃљрЃюрЃбрЃћрЃЉрЃўрЃА рЃюрЃљрЃЎрЃарЃћрЃЉрЃў рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃцрЃћрЃарЃљрЃЊ, рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџ рЃЎрЃЮрЃюрЃАрЃЮрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃА.
  * **рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ**: рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ `getSystemMemoryInfo`, `getCpuCoreCount`, `check_simd_support` рЃЊрЃљ `estimateStrassenMemoryMB` рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃћрЃю рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃБрЃџ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА.
  * **рЃЎрЃџрЃљрЃАрЃў `Matrix`**:
      * **рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў**: рЃўрЃДрЃћрЃюрЃћрЃЉрЃА `std::vector<double>`-рЃА рЃЉрЃарЃбрЃДрЃћрЃџрЃў, рЃБрЃгрЃДрЃЋрЃћрЃбрЃў рЃерЃћрЃюрЃљрЃ«рЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА.
      * **рЃЎрЃЮрЃюрЃАрЃбрЃарЃБрЃЦрЃбрЃЮрЃарЃћрЃЉрЃў**: рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃўрЃФрЃџрЃћрЃЋрЃљ рЃерЃћрЃЦрЃЏрЃюрЃљрЃА рЃњрЃљрЃюрЃќрЃЮрЃЏрЃўрЃџрЃћрЃЉрЃћрЃЉрЃўрЃЊрЃљрЃю, рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃЊрЃљрЃю рЃљрЃю `std::vector<std::vector<double>>`-рЃЊрЃљрЃю.
      * **рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў**: `+` рЃЊрЃљ `-` рЃЦрЃЋрЃћ-рЃЏрЃљрЃбрЃарЃўрЃфрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
      * **`multiply_naive`**: SIMD-рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃћрЃЉрЃБрЃџрЃў рЃАрЃљрЃЉрЃљрЃќрЃўрЃАрЃЮ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљ.
      * **`split` / `combine`**: рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА "рЃЊрЃљрЃДрЃљрЃЋрЃў рЃЊрЃљ рЃўрЃЉрЃљрЃбрЃЮрЃюрЃћ" рЃЏрЃћрЃЦрЃљрЃюрЃўрЃЎрЃљрЃА.
      * **`pad` / `unpad`**: рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃЮрЃарЃўрЃА рЃ«рЃљрЃарЃўрЃАрЃ«рЃўрЃА рЃњрЃљрЃюрЃќрЃЮрЃЏрЃўрЃџрЃћрЃЉрЃћрЃЉрЃА.
      * **I/O**: `readFromConsole`, `readFromFile`, `saveToFile`.
  * **рЃерЃбрЃарЃљрЃАрЃћрЃюрЃўрЃА рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ**:
      * **`multiplyStrassenParallel`**: рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃерЃћрЃАрЃЋрЃџрЃўрЃА рЃгрЃћрЃарЃбрЃўрЃџрЃў. рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃерЃћрЃЋрЃАрЃћрЃЉрЃљрЃА, рЃюрЃљрЃЎрЃљрЃЊрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљрЃА, рЃърЃарЃЮрЃњрЃарЃћрЃАрЃўрЃА рЃќрЃЮрЃџрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљрЃА рЃЊрЃљ рЃЏрЃБрЃерЃљрЃЎрЃўрЃА рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃљрЃА.
      * **`strassen_recursive_worker`**: рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃБрЃџрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ. рЃљрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃћрЃЉрЃА 7-рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљрЃА рЃЊрЃљ рЃўрЃерЃЋрЃћрЃЉрЃА рЃљрЃЏрЃЮрЃфрЃљрЃюрЃћрЃЉрЃА `std::async`-рЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ.
  * **рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃА рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ**:
      * **`compareMatricesParallel`**: рЃЏрЃАрЃњрЃљрЃЋрЃАрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ, рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃБрЃџрЃљрЃЊ рЃДрЃЮрЃцрЃА рЃЏрЃљрЃбрЃарЃўрЃфрЃћрЃЉрЃА рЃЊрЃљ рЃљрЃЊрЃљрЃарЃћрЃЉрЃА рЃЎрЃЋрЃљрЃЊрЃарЃљрЃюрЃбрЃћрЃЉрЃА рЃърЃљрЃарЃљрЃџрЃћрЃџрЃБрЃарЃљрЃЊ.
      * **`compareMatricesInternal`**: рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃБрЃџрЃў рЃЏрЃБрЃерЃљрЃЎрЃў рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
  * **рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃфрЃўрЃЎрЃџрЃў (`main`, `run_one_operation`)**: рЃЏрЃљрЃарЃЌрЃљрЃЋрЃА рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃА, рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃА рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљрЃА, рЃљрЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃћрЃЉрЃА рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃА, рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃА рЃЊрЃљ рЃЏрЃљрЃарЃЌрЃљрЃЋрЃА рЃџрЃЮрЃњрЃўрЃарЃћрЃЉрЃљрЃА.

## ­ЪЊі рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃўрЃА рЃЉрЃћрЃюрЃЕрЃЏрЃљрЃарЃЎрЃћрЃЉрЃў: рЃюрЃљрЃ«рЃбрЃЮрЃЏрЃў рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃЮрЃЉрЃљрЃерЃў

  * **SIMD рЃњрЃљрЃЋрЃџрЃћрЃюрЃљ:** рЃБрЃЎрЃЋрЃћ рЃерЃЌрЃљрЃЏрЃЉрЃћрЃГрЃЊрЃљрЃЋрЃў рЃЏрЃўрЃдрЃгрЃћрЃЋрЃћрЃЉрЃў рЃЎрЃўрЃЊрЃћрЃЋ рЃБрЃцрЃарЃЮ рЃФрЃџрЃўрЃћрЃарЃЊрЃћрЃЉрЃљ SIMD-рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃћрЃЉрЃБрЃџрЃў рЃАрЃљрЃЉрЃљрЃќрЃўрЃАрЃЮ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌ, рЃЌрЃБрЃЏрЃфрЃљ рЃЏрЃўрЃАрЃў рЃўрЃюрЃЊрЃўрЃЋрЃўрЃЊрЃБрЃљрЃџрЃБрЃарЃў рЃгрЃЋрЃџрЃўрЃџрЃў рЃљрЃа рЃљрЃарЃўрЃА рЃўрЃќрЃЮрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃљрЃЏ рЃЏрЃљрЃдрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА рЃбрЃћрЃАрЃбрЃћрЃЉрЃерЃў.

## ­ЪЏа№ИЈ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃў рЃЊрЃљ рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ

  * **рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃБрЃџрЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ:** Windows (рЃцрЃљрЃарЃЌрЃЮрЃЊ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА WinAPI-рЃА рЃЎрЃЮрЃюрЃАрЃЮрЃџрЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА). Linux/macOS-рЃќрЃћ рЃљрЃЊрЃљрЃърЃбрЃљрЃфрЃўрЃљ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЏрЃЮрЃўрЃЌрЃ«рЃЮрЃЋрЃА Windows-рЃАрЃърЃћрЃфрЃўрЃцрЃўрЃЎрЃБрЃарЃў рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃћрЃЉрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљрЃА.
  * **рЃЎрЃЮрЃЏрЃърЃўрЃџрЃљрЃбрЃЮрЃарЃў:** рЃАрЃљрЃГрЃўрЃарЃЮрЃљ C++17 рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ. **Microsoft Visual C++ (MSVC)** рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ `_MSC_VER` рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃћрЃЉрЃўрЃА, рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃЊрЃљ `#pragma comment(lib, "Psapi.lib")`-рЃўрЃА рЃњрЃљрЃЏрЃЮ. (рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃљрЃЊрЃљрЃърЃбрЃљрЃфрЃўрЃљ GCC/Clang-рЃўрЃАрЃЌрЃЋрЃўрЃА рЃЏрЃфрЃўрЃарЃћ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃўрЃЌ).
  * **рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃљ:** рЃЏрЃарЃљрЃЋрЃљрЃџрЃЉрЃўрЃарЃЌрЃЋрЃўрЃљрЃюрЃў CPU рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ рЃърЃљрЃарЃљрЃџрЃћрЃџрЃўрЃќрЃЏрЃўрЃА рЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃўрЃА рЃЏрЃўрЃАрЃљрЃдрЃћрЃЉрЃљрЃЊ. AVX рЃљрЃю SSE2 рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃАрЃарЃБрЃџрЃў SIMD рЃљрЃЕрЃЦрЃљрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

### рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃЎрЃЮрЃЏрЃърЃўрЃџрЃљрЃфрЃўрЃљ

1.  **рЃарЃћрЃърЃЮрЃќрЃўрЃбрЃЮрЃарЃўрЃўрЃА рЃЎрЃџрЃЮрЃюрЃўрЃарЃћрЃЉрЃљ:**

    ```bash
    git clone https://github.com/Schreiry/fluminum.git
    cd fluminum
    ```

2.  **рЃЎрЃЮрЃЏрЃърЃўрЃџрЃљрЃфрЃўрЃљ MSVC-рЃўрЃЌ (рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃў):**

      * рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ "Developer Command Prompt for VS".
      * рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ `fluminum` рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў.
      * рЃерЃћрЃљрЃАрЃарЃБрЃџрЃћрЃЌ рЃЎрЃЮрЃЏрЃърЃўрЃџрЃљрЃфрЃўрЃљ:
        ```bash
        cl fluminumTversion.cpp /EHsc /O2 /std:c++17 /Fe:fluminum.exe /link Psapi.lib
        ```
          * `/O2`: рЃарЃЌрЃљрЃЋрЃА рЃАрЃўрЃЕрЃЦрЃљрЃарЃўрЃА рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃћрЃЉрЃА.
          * `/std:c++17`: рЃљрЃДрЃћрЃюрЃћрЃЉрЃА C++ рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃА.
          * `/link Psapi.lib`: рЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃА Process Status API рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљрЃА (рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА).

### рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ рЃЎрЃЮрЃЏрЃърЃўрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃцрЃљрЃўрЃџрЃў:

```bash
./fluminum.exe
```

рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃерЃћрЃњрЃ«рЃЋрЃЊрЃћрЃЉрЃљрЃЌ рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃў рЃЏрЃћрЃюрЃўрЃБрЃЌрЃў:

1.  **рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ:** рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃљрЃдрЃЏрЃЮрЃЕрЃћрЃюрЃўрЃџ RAM-рЃА, CPU рЃЉрЃўрЃарЃЌрЃЋрЃћрЃЉрЃА рЃЊрЃљ SIMD рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљрЃА.
2.  **рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљ:** рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ рЃЏрЃљрЃбрЃарЃўрЃфрЃўрЃА рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљрЃАрЃљ рЃЊрЃљ рЃЏрЃљрЃбрЃарЃўрЃфрЃўрЃА рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃљрЃА рЃерЃЮрЃарЃўрЃА.
3.  **рЃџрЃЮрЃњрЃўрЃарЃћрЃЉрЃљ:** рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃўрЃА рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃўрЃА CSV рЃцрЃљрЃўрЃџрЃерЃў рЃџрЃЮрЃњрЃўрЃарЃћрЃЉрЃљ.
4.  **рЃњрЃљрЃюрЃќрЃЮрЃЏрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў:** рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћрЃЌ рЃЏрЃљрЃбрЃарЃўрЃфрЃўрЃА рЃќрЃЮрЃЏрЃћрЃЉрЃў.
5.  **рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљ:** рЃўрЃ«рЃўрЃџрЃћрЃЌ рЃАрЃљрЃГрЃўрЃарЃЮ RAM-рЃўрЃА рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљ.
6.  **рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃА рЃЏрЃћрЃЌрЃЮрЃЊрЃў:** рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌрЃў рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ, рЃЎрЃЮрЃюрЃАрЃЮрЃџрЃўрЃА рЃерЃћрЃДрЃЋрЃљрЃюрЃљ рЃљрЃю рЃцрЃљрЃўрЃџрЃўрЃЊрЃљрЃю рЃерЃћрЃДрЃЋрЃљрЃюрЃљ.
7.  **рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў:** рЃЊрЃљрЃљрЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃћрЃЌ рЃерЃбрЃарЃљрЃАрЃћрЃюрЃўрЃА рЃќрЃдрЃЋрЃљрЃарЃў, рЃюрЃљрЃЎрЃљрЃЊрЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљ рЃЊрЃљ (рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА) epsilon.
8.  **рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃљ:** рЃБрЃДрЃБрЃарЃћрЃЌ рЃърЃарЃЮрЃњрЃарЃћрЃАрЃўрЃА рЃќрЃЮрЃџрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃА рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћрЃЮрЃЉрЃўрЃАрЃљрЃА.
9.  **рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў:** рЃўрЃ«рЃўрЃџрЃћрЃЌ рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃўрЃА рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃљ рЃЊрЃљ рЃЊрЃарЃЮрЃћрЃЉрЃў.
10. **рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ рЃЊрЃљ рЃњрЃљрЃњрЃарЃФрЃћрЃџрЃћрЃЉрЃљ:** рЃАрЃБрЃарЃЋрЃўрЃџрЃўрЃАрЃљрЃЏрЃћрЃЉрЃа рЃерЃћрЃўрЃюрЃљрЃ«рЃћрЃЌ рЃерЃћрЃЊрЃћрЃњрЃўрЃА рЃЏрЃљрЃбрЃарЃўрЃфрЃљ рЃЊрЃљ рЃерЃћрЃљрЃАрЃарЃБрЃџрЃћрЃЌ рЃАрЃ«рЃЋрЃљ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљ.

-----

## ­Ъџђ рЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃў рЃЏрЃўрЃЏрЃљрЃарЃЌрЃБрЃџрЃћрЃЉрЃћрЃЉрЃў

рЃЏрЃўрЃБрЃ«рЃћрЃЊрЃљрЃЋрЃљрЃЊ рЃўрЃЏрЃўрЃАрЃљ, рЃарЃЮрЃЏ Fluminum рЃБрЃЎрЃЋрЃћ рЃЏрЃљрЃдрЃљрЃџ рЃЊрЃЮрЃюрЃћрЃќрЃћрЃљ рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃћрЃЉрЃБрЃџрЃў, рЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃў рЃњрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃўрЃА рЃърЃЮрЃбрЃћрЃюрЃфрЃўрЃБрЃарЃў рЃЏрЃўрЃЏрЃљрЃарЃЌрЃБрЃџрЃћрЃЉрЃћрЃЉрЃў рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА:

  * **рЃЎрЃарЃЮрЃА-рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃБрЃџрЃў рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ:** Windows API рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃћрЃЉрЃўрЃА рЃарЃћрЃцрЃљрЃЦрЃбрЃЮрЃарЃўрЃюрЃњрЃў Linux/macOS рЃЌрЃљрЃЋрЃАрЃћрЃЉрЃљрЃЊрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
  * **рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃБрЃџрЃў SIMD:** AVX-512 рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃўрЃА рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃБрЃцрЃарЃЮ рЃарЃЌрЃБрЃџрЃў рЃЋрЃћрЃЦрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃАрЃбрЃарЃљрЃбрЃћрЃњрЃўрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃгрЃљрЃЋрЃџрЃљ.
  * **GPU рЃљрЃЕрЃЦрЃљрЃарЃћрЃЉрЃљ:** рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃбрЃљрЃюрЃљ NVIDIA (CUDA) рЃљрЃю AMD/Intel (OpenCL/SYCL) GPU-рЃћрЃЉрЃќрЃћ.
  * **рЃЊрЃўрЃюрЃљрЃЏрЃўрЃБрЃарЃў рЃќрЃдрЃЋрЃљрЃарЃў:** рЃерЃбрЃарЃљрЃАрЃћрЃюрЃўрЃА/рЃюрЃљрЃўрЃЋрЃБрЃарЃў рЃќрЃдрЃЋрЃарЃўрЃА рЃЮрЃърЃбрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљ рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃўрЃАрЃљ рЃЊрЃљ рЃљрЃЏрЃЮрЃфрЃљрЃюрЃўрЃА рЃќрЃЮрЃЏрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ.
  * **Cache-Oblivious рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃћрЃЉрЃў:** рЃљрЃџрЃбрЃћрЃарЃюрЃљрЃбрЃўрЃБрЃџрЃў рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃБрЃџрЃў рЃњрЃљрЃюрЃџрЃљрЃњрЃћрЃЉрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃгрЃљрЃЋрЃџрЃљ рЃЎрЃўрЃЊрЃћрЃЋ рЃБрЃЎрЃћрЃЌрЃћрЃАрЃў рЃЦрЃћрЃерЃўрЃА рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
  * **рЃљрЃарЃљрЃЎрЃЋрЃљрЃЊрЃарЃљрЃбрЃБрЃџрЃў рЃЊрЃљ рЃљрЃарЃљрЃ«рЃљрЃарЃўрЃАрЃ«рЃЮрЃЋрЃљрЃюрЃў рЃЮрЃарЃў:** рЃБрЃцрЃарЃЮ рЃќрЃЮрЃњрЃљрЃЊрЃў рЃерЃбрЃарЃљрЃАрЃћрЃюрЃўрЃА рЃЋрЃљрЃарЃўрЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃљрЃю рЃљрЃџрЃбрЃћрЃарЃюрЃљрЃбрЃўрЃБрЃџрЃў рЃерЃћрЃЋрЃАрЃћрЃЉрЃўрЃА рЃАрЃЦрЃћрЃЏрЃћрЃЉрЃўрЃА рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ.
  * **рЃњрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃБрЃџрЃў рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃћрЃЉрЃў:** MPI рЃљрЃю рЃЏрЃАрЃњрЃљрЃЋрЃАрЃў рЃЕрЃљрЃарЃЕрЃЮрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЏрЃарЃљрЃЋрЃљрЃџ рЃЏрЃљрЃюрЃЦрЃљрЃюрЃљрЃќрЃћ рЃЏрЃљрЃАрЃерЃбрЃљрЃЉрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

## ­ЪцЮ рЃгрЃЋрЃџрЃўрЃџрЃўрЃА рЃерЃћрЃбрЃљрЃюрЃљ

рЃгрЃЋрЃџрЃўрЃџрЃўрЃА рЃерЃћрЃбрЃљрЃюрЃљ рЃЏрЃўрЃАрЃљрЃАрЃљрЃџрЃЏрЃћрЃЉрЃћрЃџрЃўрЃљ\! рЃЌрЃБ рЃњрЃљрЃЦрЃЋрЃЌ рЃўрЃЊрЃћрЃћрЃЉрЃў рЃњрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃўрЃА, рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃгрЃЮрЃарЃћрЃЉрЃўрЃА рЃљрЃю рЃљрЃ«рЃљрЃџрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ:

1.  **Fork** рЃарЃћрЃърЃЮрЃќрЃўрЃбрЃЮрЃарЃўрЃљ.
2.  рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ **рЃљрЃ«рЃљрЃџрЃў branch** рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА (`git checkout -b feature/AmazingIdea`).
3.  **рЃњрЃљрЃюрЃљрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃћрЃЌ** рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў, рЃЊрЃљрЃўрЃфрЃљрЃЋрЃўрЃЌ рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃЎрЃЮрЃЊрЃўрЃА рЃАрЃбрЃўрЃџрЃў.
4.  **рЃњрЃљрЃбрЃћрЃАрЃбрЃћрЃЌ** рЃАрЃљрЃцрЃБрЃФрЃЋрЃџрЃўрЃљрЃюрЃљрЃЊ.
5.  **рЃЊрЃљрЃљрЃЎрЃЮрЃЏрЃўрЃбрЃћрЃЌ** рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў (`git commit -m 'Add some AmazingIdea'`).
6.  **Push** рЃЉрЃарЃћрЃюрЃЕрЃќрЃћ (`git push origin feature/AmazingIdea`).
7.  рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ **Pull Request**.
